# Plateforme de formation et sensibilisation

Application React créée avec Vite.

## Installation

```bash
npm install
```

## Développement

Pour démarrer le serveur de développement sur le port 5173 :

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Build

Pour créer une version de production :

```bash
npm run build
```

## Prévisualisation

Pour prévisualiser la version de production :

```bash
npm run preview
```
